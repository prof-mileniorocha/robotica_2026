# Robótica 2026

Repositório de apoio para aulas práticas de Robótica, com exemplos em Arduino para sensores, atuadores e interfaces visuais.

## Professor

**Milênio da Rocha Marques** (perfil: [@prof-mileniorocha](https://github.com/prof-mileniorocha))

## Projetos deste repositório

Este repositório está organizado em atividades práticas:

1. **Sistema de Alarme de Emergência**
	- Pasta: `01_sistema_de_alarme_de_emergencia/`
	- Arquivo: `sistema_de_alarme_de_emergencia.ino`
	- Objetivo: demonstrar detecção/alerta em um sistema de segurança.

2. **Semáforo Inteligente**
	- Pasta: `02_semaforo_inteligente/`
	- Arquivo: `semaforo_inteligente.ino`
	- Objetivo: simular lógica de controle de tráfego com automação.

3. **Display OLED**
	- Pasta: `03_display_oled/`
	- Arquivo: `display_oled.ino`
	- Objetivo: exibir informações em tela OLED com microcontrolador.

4. **Matriz LED 8x8**
	- Pasta: `04_matriz_led/`
	- Arquivo: `matriz_led_8x8.ino`
	- Objetivo: apresentar animações e padrões visuais em matriz de LEDs.
